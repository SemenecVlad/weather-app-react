React application that gives you the ability to see your local weather with 5day/3hour forecast. Also you can find a weather in any city you want.

After cloning the repo you should create a folder named 'config' inside 'src', and add index.js file with content like that:
'export const apiKey = your api key'. You can recieve your API key on https://openweathermap.org after register.

Steps for runing project:

1) npm install or yarn install

2) npm start or yarn start
